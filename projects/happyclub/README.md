# 😊 Happy Club 😊

> Smile long enough to receive certificates and join our club!

## About the Project

Happy Club is an interactive installation exploring the social pressure to display happiness. It challenges visitors to maintain a smile for a defined period to receive a "certificate" of membership to the club. The project sits at the intersection of art and technology, using face recognition to analyze facial expressions and create a gamified experience around the concept of happiness.

The installation raises questions about authenticity, social conformity, and the digitization of human emotions - particularly in the era of social media where happiness is often performative rather than genuine.

## Technical Implementation

The Happy Club installation is built using:

- HTML5, CSS3, and JavaScript
- p5.js for canvas rendering and camera interaction
- ml5.js for facial landmark detection
- The FaceMesh model for identifying facial features

The application uses these technologies to:
1. Detect the user's face through their webcam
2. Track facial landmarks around the eyes and mouth
3. Analyze these landmarks to determine if the user is smiling
4. Provide real-time visual feedback through overlay graphics
5. Time the duration of the user's smile
6. Collect email addresses for "certificates" (no actual emails are sent)


## Running the Installation

1. Clone this repository
2. Open `index.html` in a modern web browser
3. Allow camera access when prompted
4. Follow the on-screen instructions to join the Happy Club

## Privacy Notice

This application uses your device's camera for facial detection. All processing is done locally in your browser, and no video data is transmitted or stored. If you provide an email address, it is not stored or used for any purpose.
